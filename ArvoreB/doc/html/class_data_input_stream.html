DataInputStream::DataInputStream ( char *  buffer,
int  tamanho 
)
inline

Constrói um novo objeto DataInputStream copiando todos os dados do buffer recebido para um vetor interno. Não é necessário que o buffer tenha caracteres, podem ser apenas bytes ou qualquer outra coisa. Os dados serão interpretados como um arranjo de caracteres apenas para facilitar a cópia de memória.

Parâmetros
bufferArranjo com os dados do DataInputStream.
tamanhoQuantidade de elementos do arranjo.
